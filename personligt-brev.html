Ansökan om Fullstack-utvecklare

Hej,

Jag skriver för att uttrycka mitt starka intresse för att ansluta till ert team som Fullstack-utvecklare. För närvarande slutför jag mina studier vid Teknikhögskolan och avslutar en framgångsrik LIA-praktik på FBK Balkan, där jag har byggt en solid grund inom modern webbutveckling och agil ledarskap.

Under min tid på FBK Balkan har jag varit djupt involverad i att utveckla en webbapplikation för ungdomsportalen. Det handlade inte bara om att skriva kod; jag tog på mig Scrum Master-rollen för att säkerställa att vårt team höll sig på rätt spår medan jag fokuserade på att bygga RESTful API:er med Spring Boot och hantera komplexa relationsdatabasstrukturer. Min tekniska stack – från Java och SQL på backend till React och TypeScript på frontend – gör att jag kan bygga helhetslösningar från grunden.

Bortom praktiken har jag konstant pressat mina gränser genom personliga projekt som ett bokningssystem för frisörsalonger och ett webbskrapnings-API. Jag har också ett djupt intresse för branschens framtid, specifikt inom AI-driven utveckling och bidrag till öppna källkodsprojekt som OpenClaw. Jag vill inte bara bygga appar; jag vill bygga verktyg som är intelligenta och samhällsfokuserade.

Vad jag erbjuder:

En Fullstack-tankesätt: Bekväm med att växla mellan Spring Boot-tjänster och Tailwind-stilade React-komponenter

Agilt ledarskap: Praktisk erfarenhet som Scrum Master, hantering av teamarbetsflöden och versionskontroll med Git

Anpassningsförmåga: Beprövad förmåga att snabbt lära sig och tillämpa nya teknologier för att lösa verkliga problem

Jag är ivrig att ta med min motivation och tekniska färdigheter till ert team. Tack för er tid och för att ni överväger min ansökan.

Med vänliga hälsningar,
